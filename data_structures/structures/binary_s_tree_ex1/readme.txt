Binary Search Tree:
-------------------------------
Author: Wikus Jansen van Resnburg.
Decsription: Studying a non-linear data structure.
		- This structure is implemented in C.

Fundamental operations: (The BST is always sorted based on its key value.)
	- Find a new element/node. 
	- Insert a new element/node.
	- Remove a node.
		* I:   No children,
		* II:  1 child,
		* III: 3 children.
